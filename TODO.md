# 项目进度记录

## 当前状态

海哥自媒体账号内容工作台已完成本地可运行版本和云部署准备，当前具备以下能力：

- 顶部导航与多页面工作台结构
- 双行业模式（火锅/烧烤）支持，通过 `src/industry.ts` 实现行业文本适配和配置切换
- 选题池浏览、筛选与详情查看
- 内容生产台 7 步流程推进与进度保存
- 联网调研、AI 选题建议、AI 脚本生成、AI 发布文案生成
- 本地登录页与会话校验
- 本地 Node API 代理与 `content.json` 数据写回
- 生产服务分离：`server/prod.mjs` 支持前后端一体部署
- 构建产物约 340KB，适合快速部署到云服务器

当前阶段：由”可演示原型”进入”可持续使用和可维护迭代”的产品整理阶段，同时已具备上线部署条件。
